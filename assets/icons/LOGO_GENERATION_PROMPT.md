# Prompt pour génération du logo Khadma

## Prompt principal (version complète)

```
Create a modern, professional mobile app logo for "Khadma", a service marketplace application in Morocco. 

DESIGN REQUIREMENTS:
- Circular or rounded square icon format (512x512px, suitable for mobile app)
- Clean, minimalist design that works at small sizes
- Modern flat design style with subtle gradients
- Professional and trustworthy appearance

COLOR PALETTE:
- Primary: Deep red (#C1272D) - Moroccan red
- Secondary: Forest green (#006233) - Moroccan green  
- Accent: Golden yellow (#FFC107) - Moroccan yellow
- White (#FFFFFF) for contrast elements
- Use gradients and color combinations from this palette

VISUAL ELEMENTS (choose one or combine):
- Stylized key or wrench representing services and solutions
- Handshake or helping hand symbolizing service and assistance
- Modern geometric tools (hammer, screwdriver, wrench) arranged elegantly
- Abstract connection/link symbol representing client-professional connection
- Simple letter "K" integrated with service iconography

IMPORTANT RESTRICTIONS:
- NO Moroccan star (pentagram) - avoid any star symbols
- NO political or religious symbols
- NO text in the logo itself (icon only)
- Keep it culturally neutral but inspired by Moroccan colors

STYLE:
- Flat design with subtle depth
- Bold, recognizable shapes
- High contrast for visibility
- Works in monochrome if needed
- Professional service industry aesthetic
- Modern tech app feel

TECHNICAL:
- Vector-style illustration
- Clean lines, no complex details
- Suitable for app icon, splash screen, and branding
- Scalable from 16px to 512px
- Square format with rounded corners option
```

## Prompt simplifié (version courte)

```
Modern mobile app logo icon: stylized key or tools in Moroccan colors (red #C1272D, green #006233, yellow #FFC107). Circular design, flat style, professional service marketplace aesthetic. No stars, no text, clean minimalist icon. 512x512px, suitable for app icon.
```

## Notes importantes

- **Format**: Demandez toujours une sortie carrée (1:1) pour les icônes d'app
- **Résolution**: Minimum 512x512px, idéalement 1024x1024px ou plus
- **Style**: Précisez "flat design" ou "vector style" pour un look moderne
- **Couleurs**: Mentionnez explicitement les codes hex pour précision
- **Itérations**: Testez plusieurs variantes et combinez les meilleurs éléments
